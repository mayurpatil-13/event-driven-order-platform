# Event-Driven Order Platform

## Overview
Backend-focused event-driven microservices system using OAuth2, Kafka, RabbitMQ, Redis, and Docker.

## Architecture
- API Gateway
- Auth Service
- Order Service
- Inventory Service
- Notification Worker

## Tech Stack
Node.js, .NET, Kafka, RabbitMQ, Redis, MongoDB, MySQL, Docker, Nginx
